# 团队会议倒计时器 - 用户手册

## 功能特性

### 基础功能
1. **任务管理**
   - 添加、编辑、删除子任务
   - 设置任务时间（1-180分钟）
   - 实时统计总任务数和总时长
   - 任务进度跟踪

2. **倒计时器**
   - 开始/暂停/继续/停止会议
   - 跳过当前任务
   - 为当前任务加时
   - 可视化倒计时显示

### 扩展功能 - 语音提醒
- 会议开始语音播报
- 每个任务完成语音提醒
- 会议结束语音通知
- 智能语音合成（支持中英文）

## 安装和使用指南

### 系统要求
- Python 3.6+
- Windows/macOS/Linux
- 音频输出设备

### 安装步骤
1. 安装依赖：`pip install pyttsx3==2.90`
2. 运行程序：`python main.py`
3. 点击"测试语音"确认功能正常

### 使用流程
1. **添加任务**：点击"+添加任务"，输入任务名称和时长
2. **开始会议**：点击"▶️开始会议"，系统自动语音播报开始
3. **会议进行**：系统自动倒计时并语音提醒任务完成
4. **会议控制**：可随时暂停、继续、停止或跳过任务

## 界面说明

### 主界面区域
- **任务管理区**：添加、删除、编辑任务
- **统计信息区**：显示总任务数和总时长
- **倒计时显示区**：当前任务和剩余时间
- **控制按钮区**：会议控制操作

### 按钮功能
- **+ 添加任务**：打开任务添加对话框
- **🗑️ 删除选中**：删除选中的任务
- **🗑️ 清空所有**：清空所有任务
- **📊 统计信息**：显示详细统计
- **🔊 测试语音**：测试语音功能
- **▶️ 开始会议**：开始会议倒计时
- **⏸️ 暂停**：暂停当前计时
- **▶️ 继续**：继续计时
- **⏹️ 停止**：停止会议
- **⏩ 跳过当前**：跳过当前任务
- **➕ 加时5分钟**：为当前任务增加5分钟
